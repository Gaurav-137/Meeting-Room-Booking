# 🏢 RoomSync — Meeting Room Booking

A modern, responsive meeting room booking app built with **React + Vite**.

## Features

- 📋 **Book Rooms** — Create, edit, and delete meeting bookings
- 🗓️ **Timeline View** — Visual timeline of all room bookings
- 🔍 **Search & Filter** — Filter by room, date, or keyword
- ⚠️ **Conflict Detection** — Prevents overlapping bookings
- 📅 **Custom Date & Time Pickers** — Styled calendar and time dropdowns
- 💾 **Persistent Data** — Bookings saved to localStorage

## Tech Stack

React · Vite · Vanilla CSS

## Getting Started

```bash
npm install
npm run dev
```
